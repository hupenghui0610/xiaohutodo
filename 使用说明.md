# 小胡同学 To-do List - 使用说明

## 🎉 快速开始（最简单）

**双击 `启动应用.bat` 文件**，应用会在浏览器中打开，立即可用！

## 📋 功能特点

- **无限期待办**：没有时间限制的任务
- **周待办**：本周内需要完成的任务，未完成自动顺延并标记延迟
- **日待办**：按每天管理的任务，支持周一到周五和周末分组

## 💾 数据存储

所有数据保存在浏览器的本地存储（localStorage）中：
- ✅ 完全本地，不上传任何服务器
- ✅ 关闭浏览器后数据依然保留
- ⚠️ 清理浏览器数据会删除待办记录
- ⚠️ 不同浏览器的数据是独立的

## 🖥️ 如果想要独立的桌面应用

### 方案 A：使用 Electron（功能完整，体积约 150MB）

需要先安装 Node.js：
1. 访问 https://nodejs.org/zh-cn/ 下载并安装
2. 打开命令行，运行：
   ```bash
   cd d:\cursor\cursor_workspace
   npm install
   npm run build
   ```
3. 安装包会生成在 `dist` 目录

### 方案 B：使用 WebView2（轻量，体积约 5MB）

需要先安装 .NET 6.0 SDK：
1. 访问 https://dotnet.microsoft.com/zh-cn/download/dotnet/6.0
2. 下载并安装 .NET 6.0 SDK (x64)
3. 打开命令行，运行：
   ```bash
   cd d:\cursor\cursor_workspace
   dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true
   ```
4. 可执行文件在：`bin\Release\net6.0-windows\win-x64\publish\小胡同学Todo.exe`

## 💡 推荐使用方式

**最简单**：直接用浏览器打开（双击 `启动应用.bat`），然后：
- 按 `Ctrl + D` 添加到浏览器书签
- 或右键 `index.html` -> 发送到 -> 桌面快捷方式

这样每次打开都很方便，而且不需要安装任何额外软件！

## 📞 问题反馈

如有问题，请检查：
1. 使用的是现代浏览器（Chrome、Edge、Firefox 等）
2. 浏览器版本不要太旧
3. 没有禁用 JavaScript 和本地存储
