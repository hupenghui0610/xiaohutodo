## 📋 你需要做的事情（按顺序）

### 第一步：配置飞书多维表格（5分钟）

1. **打开飞书多维表格**：https://feishu.cn
2. **找到你的多维表格**：`tblZKtePBKg6T21x`
3. **创建数据表字段**（如果还没创建）：
   - `id`（文本）
   - `type`（文本）
   - `title`（文本）
   - `done`（复选框）
   - `date`（日期）
   - `weekStart`（日期）
   - `delayed`（复选框）
   - `createdAt`（日期时间）

4. **获取数据表 ID**：
   - 打开数据表，查看 URL 中 `table=` 后面的 ID
   - 例如：`tblXXXXXXXXXXXX`

5. **修改 `api/feishu.js` 文件**：
   - 找到这一行：`const DATA_TABLE_ID = 'tblEnUS4h7MN8Yor';`
   - 替换为你的实际数据表 ID

详细步骤见：`飞书配置说明.md`

### 第二步：部署到 Vercel（10分钟）

1. **注册 Vercel**：https://vercel.com
   - 用 GitHub 账号登录

2. **导入 GitHub 仓库**：
   - 点击 "Add New..." → "Project"
   - 选择 "Import Git Repository"
   - 选择 `xiaohutodo` 仓库
   - 点击 "Deploy"

3. **配置环境变量**：
   - 进入项目 Settings → Environment Variables
   - 添加：`FEISHU_APP_SECRET` = `wgWrVSmfxVfk3HDbqsk1khnAoTG5wWI4`
   - 点击 "Save"

4. **重新部署**：
   - 进入 Deployments
   - 点击最新部署的 "..." → "Redeploy"

5. **获取 URL**：
   - 例如：`https://xiaohutodo-xxx.vercel.app`

详细步骤见：`Vercel完整部署指南.md`

---

### 第三步：配置飞书应用权限（5分钟）

1. **访问飞书开放平台**：https://open.feishu.cn/app
2. **找到你的应用**
3. **添加权限**：
   - `bitable:app`
   - `bitable:app:readonly`
   - `bitable:app:readwrite`
4. **更新桌面端主页**：
   - 填入 Vercel URL：`https://xiaohutodo-xxx.vercel.app`
5. **发布应用**：
   - 版本管理与发布 → 创建版本 → 发布
