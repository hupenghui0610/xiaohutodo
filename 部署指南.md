# 部署到 Vercel 的步骤（5分钟完成）

## 📦 准备工作
所有文件已经准备好了，就在当前目录 `d:\cursor\cursor_workspace`

## 🚀 部署步骤

### 方法 1：网页拖拽部署（最简单）

1. **注册 Vercel 账号**
   - 访问：https://vercel.com
   - 点击右上角 "Sign Up"
   - 用邮箱注册（或用 Google/GitHub 账号登录）

2. **部署应用**
   - 登录后，点击 "Add New..." → "Project"
   - 选择 "Deploy without Git"
   - 把整个文件夹 `d:\cursor\cursor_workspace` 拖拽到网页上
   - 或者点击 "Browse" 选择文件夹

3. **等待部署**
   - Vercel 会自动部署（约 30 秒）
   - 完成后会显示你的 URL，例如：`https://xiaohutodo-xxx.vercel.app`

4. **复制 URL**
   - 复制这个 URL，填入飞书后台的"桌面端主页"

---

### 方法 2：使用 Vercel CLI（如果你有 Node.js）

如果你已经安装了 Node.js，可以用命令行：

```bash
# 安装 Vercel CLI
npm install -g vercel

# 进入项目目录
cd d:\cursor\cursor_workspace

# 登录并部署
vercel

# 按提示操作，完成后会得到 URL
```
