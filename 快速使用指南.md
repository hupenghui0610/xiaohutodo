# 小胡同学 To-do List - 快速使用指南

## 🚀 立即开始使用（3 种方式）

### 方式 1：双击启动（推荐）
**双击 `启动应用.bat`** 或 **`启动应用.vbs`**，应用会在浏览器中打开。

### 方式 2：创建桌面快捷方式
**双击 `创建桌面快捷方式.vbs`**，会在桌面创建一个快捷方式，以后直接双击桌面图标即可启动。

### 方式 3：直接打开 HTML
直接双击 `index.html` 文件，用浏览器打开。

---

## ❓ 关于独立 EXE 的说明

**Windows 上无法创建完全不需要依赖的独立 EXE**，除非使用以下方案：

### 方案 A：Electron（体积约 150MB）
需要先安装 Node.js：
1. 访问 https://nodejs.org/zh-cn/ 下载安装
2. 运行命令：
   ```bash
   cd d:\cursor\cursor_workspace
   npm install
   npm run build
   ```
3. 生成的安装包在 `dist` 目录

### 方案 B：WebView2（体积约 5-10MB）
需要先安装 .NET 6.0 SDK：
1. 访问 https://dotnet.microsoft.com/zh-cn/download/dotnet/6.0
2. 运行命令：
   ```bash
   cd d:\cursor\cursor_workspace
   dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true
   ```
3. 生成的 EXE 在 `bin\Release\net6.0-windows\win-x64\publish\`

## 💡 最佳实践

**推荐使用方式 2（桌面快捷方式）**：
1. 双击 `创建桌面快捷方式.vbs`
2. 桌面会出现 "小胡同学Todo" 图标
3. 以后直接双击桌面图标启动
4. 可以右键图标 → 固定到任务栏

这样使用起来和独立应用没有区别！

---

## 📦 数据存储

- ✅ 数据保存在浏览器本地存储（localStorage）
- ✅ 完全本地，不上传任何服务器
- ✅ 关闭浏览器后数据依然保留
- ⚠️ 清理浏览器数据会删除待办记录
- ⚠️ 不同浏览器的数据是独立的

---

## 🎨 使用技巧

- 按 **F11** 切换全屏模式，获得更沉浸的体验
- 按 **Ctrl + D** 添加到浏览器书签
- 在浏览器中可以调整缩放比例（Ctrl + 滚轮）

---

## 📞 总结

**你现在就可以使用了！**

1. 最简单：双击 `启动应用.bat`
2. 最方便：双击 `创建桌面快捷方式.vbs`，然后用桌面图标启动
3. 最直接：双击 `index.html`

如果以后想要真正的独立 EXE，再安装 Node.js 或 .NET SDK 进行打包。
